# Stock App

Um aplicativo usado para gerenciar suas ações, contendo cadastro, listagem e gráficos de ações por data.

## Stacks

- React Native
- Typescript
- Expo
- Victory Charts
- AsyncStorage
- Date Picker
- Styled Components
- React Navigation

## Para rodar o projeto

Clone o projeto e rode com expo

```bash
  git clone branch_url
  cd stockapp
  npm install ou yarn install
  npx expo start
```
